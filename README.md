## 👤 Sobre Mim

Muito prazer! Sou o Breno, aluno da rede pública do Paraná. Sempre fui curioso sobre como a internet funciona e, hoje, estou mergulhado no universo do desenvolvimento. 

Meu objetivo é aprender novas linguagens de programação, dominar os bastidores de sites e sistemas, e aprimorar cada vez mais meus conhecimentos na área. Para mim, programar é como ter um superpoder para criar o que eu quiser!
